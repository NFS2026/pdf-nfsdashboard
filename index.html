EmailJS Configuration

⚠️ One-Time Setup Required

This portal uses EmailJS (free — 200 emails/month) to notify signers automatically. Follow these steps once:

1. Go to emailjs.com → Create free account → Add Email Service (Gmail)

2. Create an Email Template with these variables:

To: {{to_email}} Subject: [NFS] Action Required — {{doc_type}} {{doc_number}} awaiting your signature Dear {{signer_name}}, You have been requested to sign the following NFS document: Document: {{doc_title}} Type: {{doc_type}} | Rev: {{doc_rev}} Submitted by: {{submitted_by}} Please click the link below to review and sign: {{signing_link}} Deadline: {{deadline}} This is an automated notification from the NFS Document Signing Portal.

3. Copy your Service ID, Template ID, and Public Key below.

Google Drive Configuration

⚠️ Google Drive API Setup

1. Go to console.cloud.google.com → New Project → Enable Google Drive API

2. Create OAuth 2.0 credentials → Web Application → Add your domain to authorized origins

3. Create a folder in your Google Drive named "NFS Signing Portal" and copy its folder ID from the URL

Deployment Guide

Deploy this portal to your NFSDashboard domain in 3 steps:

① Save this file as sign.html
Place it in your GitHub repository or Netlify site root
② Complete Setup tab above
Add EmailJS and Google Drive credentials — takes ~15 minutes once
③ Access at nfsdashboard.com/sign.html
Share the URL with your team — works on any device, no install needed
